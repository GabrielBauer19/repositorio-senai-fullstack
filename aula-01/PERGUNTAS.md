# Pesquisa

## 1. O que é programação?
Programação é o processo de criar instruções para que um computador execute tarefas e resolva problemas.

## 2. O que é um algoritmo?
Um algoritmo é uma sequência de passos organizados para realizar uma tarefa ou solucionar um problema.

## 3. Qual a diferença entre algoritmo e linguagem de programação?
O algoritmo é a lógica da solução. A linguagem de programação é a forma usada para escrever essa solução para o computador entender.

## 4. O que é JavaScript?
JavaScript é uma linguagem de programação muito utilizada para criar páginas web interativas e aplicações.

## 5. Onde o JavaScript pode ser utilizado atualmente?
Pode ser utilizado em sites, servidores, aplicativos móveis, jogos e aplicações desktop.

## 6. O que é o Node.js?
Node.js é um ambiente que permite executar código JavaScript fora do navegador.

## 7. Qual é a função do Node.js no desenvolvimento com JavaScript?
Ele permite criar servidores, APIs e aplicações de back-end usando JavaScript.

## 8. O que é o VS Code?
VS Code é um editor de código-fonte desenvolvido pela Microsoft.

## 9. Qual a diferença entre um editor de texto comum e um editor de código?
Um editor de código possui recursos específicos para programação, como destaque de sintaxe, autocompletar e depuração.

## 10. O que é o terminal?
O terminal é uma ferramenta que permite executar comandos diretamente no sistema operacional.

## 11. Para que serve o comando node arquivo.js?
Serve para executar um arquivo JavaScript usando o Node.js.

## 12. O que é uma variável?
Uma variável é um espaço usado para armazenar dados que podem ser utilizados no programa.

## 13. Qual a diferença entre let e const?
`let` permite alterar o valor da variável. `const` cria uma variável cujo valor não pode ser reatribuído.

## 14. O que são tipos de dados? Cite exemplos.
Tipos de dados definem o tipo de valor armazenado. Exemplos: string, number, boolean, array e object.

## 15. Qual a principal diferença entre JavaScript e TypeScript?
JavaScript não possui tipagem estática. TypeScript adiciona tipos e recursos extras para facilitar o desenvolvimento e evitar erros.
